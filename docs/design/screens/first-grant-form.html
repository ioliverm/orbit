Tu primer grant

Introduce un grant manualmente. El calendario de vesting se dibuja a la derecha conforme escribes. Si tienes varios grants, puedes saltar este paso y usar el importador CSV más adelante.

Tipo de instrumento

Obligatorio antes de usar sell-now (Slice 5). En pre-IPO puedes dejarlo vacío.

Fechas futuras mostrarán un aviso no bloqueante.

Entero; aceptamos fracciones hasta 4 decimales para casos de ESPP / post-split.

Obligatorio si seleccionas NSO o ISO. Oculto para RSU / ESPP.

Debe ser menor o igual a los meses totales (AC-4.2.6).

Double-trigger (sólo RSU)

Déjalo vacío si aún no ha ocurrido. El timeline mostrará las acciones time-vested con trama de rayas hasta que el evento se cumpla.

La trama rayada indica acciones time-vested pendientes de evento de liquidez. Ingresos imponibles hasta la fecha: 0 acciones. Cuando registres la fecha de la IPO / adquisición / tender offer, el relleno pasará a sólido.

Sobre el algoritmo

La previsualización usa exactamente la misma función que la backend (derive_vesting_events, ADR-014 §2). Si no coinciden, el test de fixture compartido falla.

Esto no es asesoramiento fiscal ni financiero — Orbit calcula, no aconseja.
